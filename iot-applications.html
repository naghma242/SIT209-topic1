IoT Applications

An IoT Application is one that runs using IoT devices on the principal of exchanging information through the internet.
The main features needed are real-time relible connectivity for fast data transmission, security, access to surroundings, easy modifiability, and data handling ability.
Some examples would be smart devices of all types including wearables like smart watches, home appliances like TV's and smart ovens, safety equipment like smoke detectors etc.

Smart Buildings

Smart is technically referred to something that is automated, or intelligent in some way. It could mean it accepts multiple input types and is interactive.
Some of the main features needed in a smart building scenario are securrity, safety, automation, centralized control and system integration.
Some examples of smart buildings would be homes, offices and malls with automated emergency systems, security and identification systems etc.

Control Interface

Some High level features of a good control interface involve energy efficiency and sustainability, some level of AI, bidirectional and choice adaptive interactive UI, extensive record keeping abilities with real time monitoring and remote control access, among others.
The main parts needed are input and output hardware, database system for storing and fetching data, IoT device, software application and internet connectivity.
The control interface will be used by operators, clients or the target audience, owners and security personnel as well as other devices on the network.
